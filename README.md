# Car Dealership Management System

A comprehensive management system for car dealerships. This monorepo contains the Django REST API backend and the future frontend application.

## Core Features

*   **Sales Portfolio Management**: Automatically create and assign portfolios to sales representatives.
*   **Secure API**: JWT-based authentication for all endpoints.
*   **API Documentation**: Auto-generated OpenAPI 3.0 schema with Swagger UI.
*   **Historical Tracking**: Logs all changes made to critical data models.
